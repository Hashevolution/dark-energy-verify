# data/

DESI DR2 공개 데이터 보관소.

## 출처

- DESI Data Release: https://data.desi.lbl.gov/
- DR2 cosmology products: https://data.desi.lbl.gov/doc/releases/dr2/
- 관련 논문 부록의 공분산 행렬·체인.

## 정책

- 대용량 (체인, FITS) → `.gitignore`로 제외. 로컬에만 둠.
- 출처 URL·다운로드 명령은 `data/SOURCES.md`에 기록 (재현성).
- 가공된 작은 요약본 (수십 KB)만 커밋.
