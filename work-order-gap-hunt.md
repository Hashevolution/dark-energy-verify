# 작업 순서 — 검증 게이트 → 직관·데이터로 빈틈 찾기

> 목적: A(de Sitter)·B(메커니즘) 빈칸을 향해, **각 단계에 통과/탈락 게이트**를 두고
> 검증한다. 게이트에서 "기존 이론과 안 겹치는" 결과가 나오는 자리 = **빈틈 후보**.
> 핵심 원칙(핸드오버 §9.15): 흥분되는 패턴은 *측정 방식 바꿔* 자가반증부터. 살아남으면 신호.
> 작성일: 2026-06-16. 환경: numpy/scipy 설치됨, Pantheon+ 네트워크 접근 가능(확인).

---

## 0. 전체 흐름 (한눈)

```
Phase 1 (데이터)   과제 C: de Sitter 신호 진위    → A의 표적이 실재하나
Phase 2 (수치)     magic 축 추가 스핀사슬          → 노트가 놓친 변수 있나 (B 신규)
Phase 3 (수식)     모듈러-RG / 인식-가중 상호정보  → c-정리 초과항 있나 (B 판정)
Phase 4 (종합)     어느 게이트가 "안 겹침" 냈나    → 그 자리에 직관+데이터 집중
```

원칙: **탈락 = 실패가 아님.** 탈락 양상이 두 가지로 갈림 —
- (a) "또 기존과 겹침/아티팩트" → 빈틈 아님, 다음 Phase로.
- (b) "기존 법칙으로 설명 안 되는 잔차 생존" → **빈틈 후보. Phase 4로 승급.**

## Phase 1 — 데이터 트랙 〔과제 C, 핸드오버 1순위〕

**검증 대상:** 초신성(Pantheon+) 분석의 **선택(컷·prior·보정)** 이 우주론 신호
(H₀, 암흑에너지 $w(z)$)를 얼마나 흔드나 = de Sitter 동역학 신호의 진위.

| 단계 | 방법 | 산출 |
|---|---|---|
| 1.1 | Pantheon+ SH0ES 공개 데이터/공분산 로드, 표준 ΛCDM 적합 재현 | $H_0,\Omega_m$ 기준값 |
| 1.2 | 컷(저z 제외, 호스트질량 보정 on/off, 광곡선 보정 변형) 스캔 | 각 선택별 $H_0,w$ |
| 1.3 | $w_0w_a$CDM 적합으로 DESI식 동역학 암흑에너지 신호 재측정 | $w_0,w_a$ 민감도표 |

**게이트 1:** 신호가 선택에 **강건**한가?
- 강건 → de Sitter 이탈(동역학 $w$) 신호 *실재* = **과제 A가 겨냥할 표적 확보.**
- 흔들림 → 신호 = 분석 아티팩트 → 빈틈 아님(걸러냄). 그래도 "어떤 선택이 흔드나"가 산출.

**왜 1순위:** A에 *데이터로* 붙는 유일한 다리. 이론(A) 단독 돌파는 최전선 합류라 느림.

## Phase 2 — 수치 트랙 〔신규 1순위, B의 새 부품〕

**검증 대상:** 노트의 모든 스핀사슬 작업(§9.8~9.15)은 **얽힘 한 변수**만 봤다.
2026.06 분야 합의 = "중력엔 얽힘 + **magic(비안정자성)** 두 변수 필요."
→ **곡률 다이얼 × {얽힘, magic} 2D 훑기.** magic이 얽힘과 *독립적으로* 곡률에 반응하면,
노트가 놓친 변수 = 빈틈 후보.

| 단계 | 방법 | 산출 |
|---|---|---|
| 2.1 | TFIM/레인보우(비균일 XX) 사슬 바닥상태 numerics 스캐폴드 작성, §9.9 얽힘 봉우리·$R=-h^2$ 재현 | sanity check 통과 |
| 2.2 | **stabilizer Rényi entropy**(magic 측도) 구현, 같은 상태에서 측정 | magic(s) 곡선 |
| 2.3 | 곡률 $h$ 9점 훑기 × {얽힘 $S$, magic $M$} 동시 측정 | 2D 응답표 |
| 2.4 | 자가반증: 측도/정규화 바꿔 magic 응답이 살아남나(§9.15 교훈) | 강건성 판정 |

**게이트 2:** magic $M(h)$가 얽힘 $S(h)$로 **환원 안 되는** 곡률 응답을 보이나?
- 독립 응답 생존 → **노트가 못 본 축. 가장 강한 빈틈 후보 → Phase 4.**
- $S$에 묶임/측도 아티팩트 → 또 재발견. 다음으로.

**왜 신규 1순위:** 노트가 *아직 안 만진* 가장 가까운 좌표. 자기완결(외부 데이터 불요).

> **✅ 실행 (2026-06-16) → ⚠️ 문헌 정독 후 판정 정정.**
> `phase2_magic_curvature.py`/`phase2b_selffalsify.py`: 원시 신호(magic 내부봉우리 ≠ 얽힘
> 단조) 떴고, 자가반증 T2에서 봉우리가 임계점 추종처럼 보여 1차 판정 (a) 재발견.
> **그러나 원논문 정독 결과 1차 판정 철회:** 우리가 잰 *전체* stabilizer Rényi 엔트로피는
> 중력-관련 양이 아님. Cao et al.(arXiv:2306.14996)·backreaction-is-magical(2403.07056)·
> magic-enriched codes(2603.13475)는 중력에 **비국소(nonlocal) magic** 을 요구; 전체 SRE는
> 임계도 탐침으로도 불신(arXiv:2501.12146). → **재판정: 측도 오류로 INCONCLUSIVE.**
> **Phase 2′:** 곡률 컷 가로지르는 *비국소* magic(상호 SRE, arXiv:2601.13314 / 부피법칙
> 보정항)으로 재측정. 비국소 magic은 얽힘 컷에 묶여 노트 그림에 더 가까움 → magic 가설
> 되살아남. 상세·근거: `experiments/phase2_RESULTS.md`.

## Phase 3 — 수식 트랙 〔B 판정, §9.2①③〕

**검증 대상:** 인식-가중(=Rényi $\alpha=1-r$) 상호정보 $I_s(A{:}B)$의 스케일 정점 $s_*$가
**c-함수 단조성으로 다 설명되는가**, 아니면 초과항이 남는가.

| 단계 | 방법 | 산출 |
|---|---|---|
| 3.1 | Phase 2 사슬에서 스케일별 $I_s(A{:}B)=S_A+S_B-S_{AB}$ 계산, $\alpha=1-r$ 가중 | $I_s(s,r)$ |
| 3.2 | RG c-함수(중심전하 흐름) 추출, $I_s$ 정점이 c-단조와 분리되나 | 초과항 유무 |
| 3.3 | (선택) 모듈러-RG 고리 holonomy 수치 근사 → a-정리 $\Delta a$ 초과 점검 | B 판정식 |

**게이트 3:** 잔차가 a-정리 $\Delta a=a_{UV}-a_{IR}$ 로 다 설명되나?
- 다 설명 → A 명제는 해석으로 후퇴(새 물리 아님).
- 초과항 생존 → 빈틈 후보 → Phase 4. (단 학계 미정리 영역, 난도 최상.)

## Phase 4 — 빈틈 종합 〔직관 + 데이터 집중〕

Phase 1~3 게이트 중 **(b) 잔차 생존**이 나온 자리만 모은다. 그 자리에:
- 직관(노트의 그림)으로 *왜 거기서 안 겹치는지* 서술,
- 데이터/수치로 *재현·강건성* 재확인(자가반증 1회 더),
- 살아남으면 그것이 arXiv 한 편의 씨앗 / 살아있는 최전선에 던질 질문.

**예상 1순위 빈틈(가설):** Phase 2의 magic 독립 응답. 근거 — 노트의 자가반증 3회는 전부
*얽힘 한 축*에서만 측정해 떨어졌다. magic 축은 그 측정에 *원리적으로 안 잡히던* 자유도.
